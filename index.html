Project 1: Basic Webpage

Overview

For this project, you will create a basic webpage using HTML to practice what you've learned about HTML structure, semantic markup, and organizing content. The goal is to create a simple webpage that effectively uses headings, paragraphs, lists, and links to create a coherent and well-structured page.

Considerations:

  • Semantic Markup: Use appropriate HTML elements to ensure semantic markup and proper structuring of content.
  • Organization: Organize your content logically and hierarchically using headings and paragraphs.
  • Readability: Consider font choices, text alignment, and spacing to enhance the readability of your webpage.
  • Consistency: Maintain a consistent design and layout throughout the webpage.
  • File Management: Ensure your files are organized in a structured folder system.
